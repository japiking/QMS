테스트 서버로 거래 전송 하는 방법 [ CA31 : 개발 , CA81 : 테스트 ]
1. 프레임워크 접속후 > 클라이언트 메뉴 관리 > 개발공통 > Property DB 관리 화면 접속
2. 프로퍼티 그룹 ID 가 was_{INSTANCE} ( was_인스턴스 ) 의 상세보기 를 클릭
3. 프로퍼티 ID 에 BIZ_WAS_CONFIG 로 조회
4. WAS별 설정 클릭 : 인스턴스가 '기업로컬'의 설정된값을 CA81 로 변경후 저장
5. 로컬 서버 재기동
6. 원래값인 CA31 로 다시 변경후 저장
