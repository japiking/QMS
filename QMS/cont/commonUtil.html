commonUtil.makeSelectBox( id, dataList, key, val, fn, selectedValue, isClear )

입력 받은 ID 밑에 dataList 로 option 을 생성한다.
  @param id (필수)콤보가 그려질 select 엘리먼트의 id
  @param dataList (필수)콤보를 그릴 데이터 json array
  @param key (옵션)option 엘리먼트의 value에 셋팅 될 데이터의 키 ( default : "code" )
  @param val (옵션)option 엘리먼트의 text에 셋팅 될 데이터의 키 ( default : "value" )
  @param fn (옵션)text의 포멧팅 함수 parma1:로우데이터, param2:value
  @param selectedValue (옵션)selected 될 값
  @param isClear (옵션:default=false)대상 select 에 기존 option 항목의 삭제 여부. var dataList = [{ "key" : "123", "val" : "일이삼" }, { "key" : "345", "val" : "삼사오" }]; commonUtil.makeSelectBox( "selectBoxID", dataList ); commonUtil.makeSelectBox( "selectBoxID", dataList, "key", "val" ); commonUtil.makeSelectBox( "selectBoxID", dataList, "", "", null, "123" ); //123 기본 선택 된 상태서 만들어짐 commonUtil.makeSelectBox( "accNum", data["_BeginLoop_F"], "code", "value", function(r,v){ return formatUtil.kebAcctNo(v, r["BANK_CODE"]); // v 는 현재 인자값(Cell), r 은 해당 로우 데이터 전체 }, "기본선택값"); * 콤보를 그릴 데이터 리스트 외에 <option>항목에 아래와 같이 '전체' 또는 '선택' 등의 항목을 추가 하고자 할경우 아래처럼 html 코드에 추가할 항목을 미리 선언해놓고 commonUtil.makeSelectBox 함수를 호출합니다. <select title="타이틀" style="width:160px;" class="select" id="test"> <option value="">전체</option> </select> *기존에 그려져있는 옵션항목을 삭제후 셋팅할 필요가 있을경우 commonUtil.makeSelectBox 함수의 7번째 인자에 true 값을 셋팅한다.
