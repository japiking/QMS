1. Framework 등록

1.1. 공통

  • 기존 I로 시작하는 아이디를 B로 등록한다.
  • 예) 기존 서비스가 keb.ibs.co.ICO01011.web 일 경우
         BOL.VBCO01011.web 으로 등록한다.

1.2. View

  • 화면을 호출하는 서비스일 경우 접두어 BOL.V 를 붙여서 등록한다.
  • 예) 기존 서비스가 keb.ibs.co.ICO01011.web 일 경우
         BOL.VBCO01011.web 으로 등록한다.

1.3. Action

  • 액션(거래)을 호출하는 서비스일 경우 접두어 BOL.A 를 붙여서 등록한다.
  • 예) 기존 서비스가 keb.ibs.co.ICO01011.web 일 경우
         BOL.ABCO01011.web 으로 등록한다.

1.4. 거래(in/out)

  • Ajax 거래를 위한 action 의 입출력 항목 정의를 위한 아이디는 .web을 제외한 action의 아이디로 등록한다.
  • 예) action 아이디가 BOL.ABCO01011.web 일경우
          BOL.ABCO01011 으로 등록한다.


2. JSP, Javascript

2.1. JSP

  • 기존 I로 시작하는 파일명을 B*****.jsp로 만든다.
  • 예) /ibsproject/app/webApp/jsp/ibs/dp/IDP0102P.jsp => /ibsproject/app/webApp/BOL/jsp/ibs/dp/BDP0102P.jsp
    예) /ibsproject/app/webApp/xui/ibs/at/IAT0060X.xml => /ibsproject/app/webApp/BOL/jsp/ibs/at/BAT0060X.jsp

2.2. JSP 접미사

  • P : 일반
  • X : 실행(보안카드, 인증서 제출등의 실행을 하는 화면)
  • U : 보고서
  • D : 팝업
  • * jsp 파일의 접미사로는 위 항목중에 해당 화면의 기능에 맞는 접미사를 붙여서 생성한다.

2.3. Javascript

  • 해당 javascript를 사용할 jsp 파일명과 같은 이름으로 /ibsproject/app/webApp/BOL/js/ibs/업무디렉토리 에 만든다.
  • 예) JSP 파일 : /ibsproject/app/webApp/BOL/jsp/ibs/dp/BDP0102P.jsp
          javascript 파일 : /ibsproject/app/webApp/BOL/js/ibs/dp/BDP0102P.js
